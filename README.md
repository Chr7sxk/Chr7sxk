### [🛡️ Wazuh SIEM Threat Detection Lab](https://github.com/Chr7sxk/wazuh-siem-lab)
Deployed a fully isolated 3-VM environment (Wazuh 4.14.5 + Kali Linux 2026.1 + Metasploitable2) to simulate real-world attacks and detect them end-to-end using a self-configured SIEM.

- Captured **300+ live events** — brute force attempts, auth failures, and anomaly detections

- Authored **custom XML detection rules** that triggered rule 100002 at **level 12 (high severity)**
- Validated the full detection chain: attack execution → log ingestion → SIEM alert

Built a fully air-gapped attack/defense environment (VirtualBox + Kali + Metasploitable2) with zero network exposure to practice real penetration testing methodology.

- Enumerated **20+ open services** (FTP, HTTP, MySQL, VNC, IRC) via Nmap reconnaissance
